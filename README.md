# 🛍 Projeto E-Commerce Web (HTML, CSS e JavaScript)

Projeto desenvolvido com a finalidade de praticar conceitos de Front-End aplicado à área de vendas digitais

## 🚀 Sobre o projeto
O projeto visa a criação de um site útil para vendas digitais, tirando do papel um projeto de loja.

## 🛠️ Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## 📚 Aprendizados
- Estruturação de páginas com HTML
- Estilização com CSS
- Noções de desenvolvimento web
- Noções de design digital e apresentação ao cliente
- Manipular elementos com o DOM
- Criação de múltiplas guias dentro de um projeto

## ▶️ Como visualizar
Abra o arquivo `index.html` no navegador.
